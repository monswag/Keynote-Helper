# Keynote-Helper

中文 | [English](README_en.md)

整理了一些自己使用 Keynote 的方法和窍门，主要包含排版、构件、动画这些内容。其中的项目文件可以直接添加至你的文件当中，关于构件和动画的使用方式会有单独的说明，我会慢慢补充。希望这个项目可以在你制作 Keynote 时提供一些帮助。
