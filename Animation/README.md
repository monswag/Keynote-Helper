# 构件动画使用实例

恰到好处的动画效果能够更好的服务于演示，希望通过这些实例帮助你更快的找到合适的动画效果，而不是在 Keynote 中一一预览。当然，这里的实例难以覆盖所有实际场景，而通过组合以及细微的调整你也许可以找到更好的解法，这和编写程序有些类似。
